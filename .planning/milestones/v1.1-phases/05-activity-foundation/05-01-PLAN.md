    - findById returns matching item or undefined

    ActivityLog.startEvent:
    - Returns event with auto-increment id (first call id=1, second id=2)
    - Event has status "pending", result null, durationMs null, all tx fields null
    - Event has agentId, agentAddress, toolName, params from input
    - Event has timestamp (number, close to Date.now())
    - Event is retrievable via getAll()

    ActivityLog.completeEvent:
    - Updates event status to "success" or "error"
    - Sets result and durationMs (durationMs > 0)
    - Non-existent id is silently ignored (no throw)

    - Sets txHash, txStatus, blockNumber on existing event
    - Does NOT trigger listener notification

    ActivityLog.onEvent listener:
    - Listener called on startEvent with the pending event
    - Listener called on completeEvent with the updated event
    - Listener NOT called on enrichEvent
    - Unsubscribe function removes listener (no further calls)

    BigInt safety:
    - Params containing BigInt values are serialized safely (no TypeError)

    globalThis singleton:
    - getActivityLog() returns same instance on repeated calls

      - `private buffer: (T | undefined)[]` pre-allocated to capacity
      - `private head = 0` (next write position)
      - `private count = 0`
      - `push(item: T)`: write at head, advance head modulo capacity, increment count (cap at capacity)
      - `getAll(): T[]`: iterate from oldest to newest, return array
      - `findById(predicate: (item: T) => boolean): T | undefined`: linear scan
      - `get size(): number`

      - `private buffer = new CircularBuffer<ActivityEvent>(capacity)` (default 500)
      - `private nextId = 1`
      - `private listeners = new Set<(event: ActivityEvent) => void>()`
      - `startEvent(data: { agentId, agentAddress, toolName, params })`: Create event with pending status, null result/durationMs/tx fields, timestamp from Date.now(). Serialize params with BigInt replacer: `JSON.parse(JSON.stringify(params, (_, v) => typeof v === 'bigint' ? v.toString() : v))`. Push to buffer, notify listeners, return event.
      - `completeEvent(id, update: { result, status })`: Find event by id, update result/status/durationMs (Date.now() - timestamp), notify listeners. Silent no-op if not found.
      - `enrichEvent(id, tx: { txHash, txStatus, blockNumber })`: Find event by id, set tx fields. NO notify. Silent no-op if not found.
      - `getAll(): ActivityEvent[]`: delegates to buffer.getAll()
      - `onEvent(callback): () => void`: Add to listeners set, return unsubscribe function that deletes from set.
      - `private notify(event)`: iterate listeners, call each in try/catch (swallow errors).
