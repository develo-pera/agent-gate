Hook signature: `useActivitySSE(enabled: boolean = true)` — the `enabled` param allows disabling SSE (useful during demo mode).

Implementation:
- State: `events` as `ActivityEvent[]` (newest-first), `isConnected` boolean
- On mount (when enabled=true): create `EventSource` pointing to `/api/activity/sse`
- Listen for `"activity"` named event (not default `message`): `es.addEventListener("activity", handler)`
- Handler: parse `e.data` as `ActivityEvent`, deduplicate by `id`:

- Set `isConnected = true` on `open` event
- Set `isConnected = false` on `error` event
- Reconnection: native EventSource auto-reconnects. Store `lastEventIdRef` from each event for reference.
- Cleanup: `es.close()` on unmount or when `enabled` changes to false
- Return `{ events, isConnected, clearEvents }` where `clearEvents` resets the events array to `[]`
- Export `useActivitySSE`
