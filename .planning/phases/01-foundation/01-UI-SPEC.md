# Phase 1 — UI Design Contract

> Visual and interaction contract for the Foundation phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn (initialize during plan execution) |
| Preset | Default dark -- customize via CSS variables below |
| Component library | Radix UI (via shadcn) |
| Icon library | lucide-react |
| Font | Inter (variable, via next/font/google) |

**shadcn initialization note:** Run `npx shadcn@latest init` in the `app/` package during plan execution. Set style "default", base color "zinc", CSS variables ON. Then override CSS variables per the Color section below.

**Components to install for Phase 1:**
- `button` -- wallet connect CTA, nav actions
- `avatar` -- wallet address display
- `badge` -- Base network badge, demo mode indicator
- `tooltip` -- sidebar icon labels, disabled-state hints
- `separator` -- sidebar dividers
- `sheet` -- mobile-irrelevant but useful for sidebar expand pattern
- `skeleton` -- loading states on placeholder pages

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, icon rail padding |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, sidebar width gutters |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Page-level vertical spacing |
| 3xl | 64px | Not used in Phase 1 |

Exceptions:
- Sidebar icon rail collapsed width: 56px (icon 24px + 16px horizontal padding)
- Sidebar expanded width: 240px
- Demo mode top banner height: 40px
- Minimum touch target for sidebar icons: 44px square

## Typography

**Inter scale (3 sizes, 2 weights):**

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body / Label | 14px | 400 (regular) for body, 600 (semibold) for labels | 1.5 | Default text, descriptions, banner copy, sidebar expanded labels, badge text, network name |
| Heading | 20px | 600 (semibold) | 1.2 | Page titles, "Coming soon" card headings |
| Display | 28px | 600 (semibold) | 1.15 | AgentGate wordmark in sidebar |

**Weights used:** 400 (regular) and 600 (semibold) only. No other weights permitted.

- Body text, descriptions, placeholder subtitles: 14px / 400
- Labels, badge text, sidebar nav labels: 14px / 600
- Headings, card titles: 20px / 600
- Wordmark display: 28px / 600

**Font stack:** `Inter, ui-sans-serif, system-ui, -apple-system, sans-serif`

**Monospace (addresses -- separate font stack, not counted in Inter scale):** `font-mono` (Tailwind default: `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace`) at 13px weight 400 for wallet addresses and hex values.

## Color

### CSS Variable Overrides (applied to `:root` dark theme)

| Role | CSS Variable | HSL Value | Hex Approx | Usage |
|------|-------------|-----------|------------|-------|
| Dominant (60%) | `--background` | 222 47% 8% | #0B0F1A | Page background, root surface |
| Secondary (30%) | `--card` | 222 40% 12% | #121929 | Cards, sidebar background, nav surface |
| Card border | `--border` | 222 30% 18% | #1E2740 | Glassmorphism card borders |
| Muted surface | `--muted` | 222 30% 16% | #1A2236 | Hover states, subtle fills |
| Muted foreground | `--muted-foreground` | 222 15% 55% | #808BA3 | Secondary text, descriptions |
| Foreground | `--foreground` | 220 15% 90% | #DFE3EB | Primary text |
| Accent | `--primary` | 270 95% 65% | #8B5CF6 | See reserved-for list below |
| Accent hover | `--primary` lightened | 270 95% 72% | #A78BFA | Hover state on accent elements |
| Accent glow | n/a (box-shadow) | 270 95% 65% @ 25% opacity | -- | Glow effect on active/focused elements |
| Destructive | `--destructive` | 0 72% 51% | #DC2626 | Destructive actions only |
| Success | custom `--success` | 142 72% 45% | #16A34A | Transaction confirmations, connected status |
| Warning | custom `--warning` | 38 92% 50% | #F59E0B | Demo mode banner accent |

### Accent Reserved For (10% rule -- these elements ONLY)

1. Primary CTA buttons (Connect Wallet, Execute)
2. Active sidebar navigation item indicator (left border or icon fill)
3. AgentGate wordmark glow effect
4. Focus ring on interactive elements (`ring-primary`)
5. Badge accent variant (Base network badge)
6. Link text on hover

**NOT accent:** card backgrounds, borders at rest, body text, section backgrounds.

### Glassmorphism Recipe

```css
.glass-card {
  background: hsl(222 40% 12% / 0.6);    /* --card at 60% opacity */
  backdrop-filter: blur(16px);
  border: 1px solid hsl(222 30% 18% / 0.5); /* --border at 50% opacity */
  border-radius: 12px;
}
.glass-card--glow {
  box-shadow: 0 0 24px hsl(270 95% 65% / 0.15); /* accent glow */
}
```

## Interaction States

### Sidebar Navigation
| State | Behavior |
|-------|----------|
| Default | Collapsed (56px), icons only |
| Hover (sidebar) | Expands to 240px over 200ms ease-out, labels fade in |
| Mouse leave | Collapses to 56px over 200ms ease-in |
| Nav item click | Route change, active indicator updates immediately |
| Nav item hover | Background fill `--muted`, icon color transitions to foreground |

### Wallet Connect Flow
| State | Behavior |
|-------|----------|
| Disconnected | "Connect Wallet" button visible in sidebar bottom |
| Connecting | RainbowKit modal opens (managed by RainbowKit) |
| Connected | Button replaced with avatar + address + network badge, demo banner disappears |
| Error | RainbowKit handles error display within its modal |

### Demo Mode
| State | Behavior |
|-------|----------|
| Active (default) | Banner visible, all pages use hardcoded treasury address for reads |
| Wallet connected | Banner disappears, app switches to connected wallet address |
| Wallet disconnected | Banner reappears, app reverts to demo mode |

## Accessibility Baseline

| Requirement | Implementation |
|-------------|---------------|
| Color contrast | All text meets WCAG AA (4.5:1 body, 3:1 large). Foreground #DFE3EB on background #0B0F1A = ~13:1. Muted-foreground #808BA3 on card #121929 = ~4.8:1. |
| Focus indicators | 2px ring in accent color (`ring-2 ring-primary ring-offset-2 ring-offset-background`) on all interactive elements |
| Keyboard navigation | Tab order: sidebar nav items top-to-bottom, then main content. Sidebar expandable via keyboard focus (not hover-only -- add focus-within trigger) |
| ARIA labels | Sidebar nav: `<nav aria-label="Main navigation">`. Each nav item: accessible name matching label text. Demo banner: `role="status"` |
