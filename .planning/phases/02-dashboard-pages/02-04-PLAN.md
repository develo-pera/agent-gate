**Form validation per UI-SPEC Interaction Patterns:**

- Submit button disabled until all 3 fields valid and not loading.
- Min button height 44px.

**Submit handler:**
```typescript
const handleSubmit = async () => {
  const res = await execute({
    delegate_address: delegateAddress,
    scope,
    max_amount: parseFloat(maxAmount),
  }, dryRun);

  setShowResult(true);

  // If not dry-run and successful, add to local delegation list
  if (!dryRun && !isDemo) {
    addDelegation({
      id: crypto.randomUUID(),
      delegate: delegateAddress,
      scope,
      caveats: { maxAmount, token: "wstETH" },
      status: "active",
      createdAt: new Date().toISOString(),
    });
    onOpenChange(false);
  }
};
```

- If `isDemo`: show `<p className="text-xs text-muted-foreground">Dry-run only in demo mode. Connect a wallet to execute transactions.</p>`

- If `error`: show `<p className="text-xs text-destructive">{error}</p>`

Sheet side: "right" (default for Sheet). Width: use default or `className="sm:max-w-md"` on SheetContent.
