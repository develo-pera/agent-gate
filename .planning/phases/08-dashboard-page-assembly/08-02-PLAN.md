- `nextIdRef: useRef<number>` initialized to `10000` (high IDs avoid collision with real events)

`startDemo` callback:
- Guard: if `agents.length === 0` or `isRunning`, return
- Set `isRunning = true`
- Use `let index = 0` counter
- Use `setInterval` with 2500ms (2.5 seconds) interval:
  - If `index >= DEMO_SEQUENCE.length`: `clearInterval`, return (keep isRunning true, chip stays)
  - Get step = `DEMO_SEQUENCE[index]`
  - Pick agent: `agents[step.agentIndex % agents.length]`
  - Create full `ActivityEvent` with:
    - `id: nextIdRef.current++`
    - `agentId: agent.agent_id`
    - `agentAddress: agent.address`
    - `toolName: step.toolName`
    - `params: step.params`
    - `result: step.result`
    - `status: step.status`
    - `timestamp: Date.now()`
    - remaining fields from step
  - Prepend to `demoEvents` state (newest first)
  - `index++`
- Store interval in `intervalRef` for cleanup

Cleanup on unmount: clear interval via `useEffect` return

Return: `{ demoEvents, isRunning, startDemo }`

Merged events (useMemo):
```typescript
const allEvents = useMemo(() => {
  const merged = [...sseEvents, ...demoEvents];
  // Deduplicate by id (SSE events take precedence over demo events)
  const seen = new Map<number, ActivityEvent>();
  for (const event of merged) {
    const existing = seen.get(event.id);
    if (!existing || (existing.status === "pending" && event.status !== "pending")) {
      seen.set(event.id, event);
    }
  }
  // Sort newest first
  return Array.from(seen.values()).sort((a, b) => b.timestamp - a.timestamp);
}, [sseEvents, demoEvents]);
```

Track new event IDs for glow effect:
```typescript
const prevCountRef = useRef(0);
const [newIds, setNewIds] = useState<Set<number>>(new Set());

useEffect(() => {
  if (allEvents.length > prevCountRef.current) {
    const newOnes = allEvents.slice(0, allEvents.length - prevCountRef.current).map(e => e.id);
    setNewIds(new Set(newOnes));
    // Clear new status after 2 seconds
    const timer = setTimeout(() => setNewIds(new Set()), 2000);
    prevCountRef.current = allEvents.length;
    return () => clearTimeout(timer);
  }
}, [allEvents]);
```

Derive agent statuses from merged events (client-side, per CONTEXT.md decision):
```typescript
const enrichedAgents = useMemo(() => {
  if (!agents) return [];
  return agents.map((agent) => {
    const agentEvents = allEvents.filter((e) => e.agentId === agent.agent_id);
    const hasActivity = agentEvents.length > 0;
    const hasPending = agentEvents.some((e) => e.status === "pending");
    const lastEvent = agentEvents[0]; // newest first
    return {
      ...agent,
      status: hasPending ? "active" as const : hasActivity ? "idle" as const : "registered" as const,
      lastAction: lastEvent?.toolName,
    };
  });
}, [agents, allEvents]);
```
